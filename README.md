# Dashboard DRE - SG Global Group

Projeto de análise financeira desenvolvido em Power BI com foco em visualização de resultados (DRE).

## 🔍 O que foi feito
- Unificação de bases Brasil e EUA
- Conversão cambial
- Modelagem de dados
- Criação de KPIs (Receita, EBITDA, Lucro)
- Dashboard interativo

## 🛠️ Ferramentas
- Power BI
- Power Query
- DAX

## 📊 Resultados
- Margem EBITDA acima de 60%
- Identificação de principais custos
- Consolidação financeira internacional
